# React Web Music
## 技术栈
- react18 + ts
- styled-components
- axios
- rtk
- react-router 6
- ant-design

## 功能
- 首页 
  - 推荐页
    - 轮播图区域
    - 歌单区域
    - 新碟上架区域
    - 新歌区域
    - MV区域
    - 排行榜区域
  - 歌手页
  - 歌单页
  - 新碟上架页
  - 排行榜页
- 播放页
  - 歌曲播放
  - 上一首/下一首切换
  - 歌词滚动
  - 播放列表删除与清空
